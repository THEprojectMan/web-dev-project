Animatio 1: Bounce

Brief explanation: The bounce animation in CSS creates an animation effect that causes an element to appear as if it's bouncing up and down on the screen. This is achieved by defining a series of keyframes that specify the element's position at various points in time. The animation is then applied to the element using CSS animation properties.
